# 🚀 Dynamic Portfolio

A fully responsive, modern portfolio website showcasing your professional profile, achievements, skills, and projects. Built with **vanilla HTML5, CSS3, and JavaScript** featuring smooth animations, dark mode toggle, and interactive elements.

## ✨ Features

### 🎨 Design & Animations
- **Smooth animations** - Slide-in effects, floating elements, and hover transitions
- **Dark/Light mode toggle** - Theme preference stored in localStorage
- **Fully responsive** - Optimized for mobile, tablet, and desktop screens
- **Modern UI** - Gradient backgrounds, glassmorphism effects, and rounded corners
- **Progress bar** - Visual scroll indicator at the top

### 📱 Sections
1. **Navigation** - Fixed navbar with smooth scroll links and theme toggle
2. **Hero Section** - Eye-catching introduction with profile image and CTA buttons
3. **About Me** - Personal introduction with key information
4. **Achievements** - Showcase accomplishments with icons and badges
5. **Skills** - Animated skill bars with proficiency levels
6. **Projects** - Featured projects with tech stacks and links
7. **Contact** - Email, phone, location details and contact form
8. **Footer** - Copyright and social credits

### 🎯 Interactive Features
- **Navigation highlighting** - Active section indicator
- **Skill bar animations** - Animated on scroll using Intersection Observer
- **Form validation** - Contact form with field validation
- **Social links** - Quick access to LinkedIn, GitHub, and resume
- **Mobile menu** - Hamburger menu for smaller screens
- **Smooth scrolling** - CSS `scroll-behavior: smooth`
